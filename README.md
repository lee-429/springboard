# SpringBoard

Spring Boot 기반 REST API 게시판 서비스 프로젝트

## 📌 Project Overview

Spring Boot를 활용하여 게시판 서비스를 구현한 백엔드 프로젝트입니다.

회원 인증부터 게시글/댓글 관리, 검색, 페이징, 파일 업로드 등
실제 웹 서비스에서 필요한 백엔드 기능 구현을 목표로 개발했습니다.

---

## 🛠 Tech Stack

### Backend
- Java 21
- Spring Boot
- Spring Data JPA
- Spring Security
- JWT
- MySQL
- Gradle

### Database
- MySQL

---

## ✨ Features

### Member
- 회원가입
- 로그인
- JWT 기반 인증/인가
- Access Token / Refresh Token 처리

### Board
- 게시글 CRUD
- 게시글 검색
- 페이징 처리
- 파일 업로드

### Comment
- 댓글 CRUD
- 작성자 권한 검증

### Security
- Spring Security 적용
- JWT Authentication Filter 구현
- 사용자 권한 관리
